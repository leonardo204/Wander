# Wander

**여행 사진을 타임라인으로, AI가 스토리로**

Wander는 여행 사진의 GPS와 시간 정보를 분석하여 자동으로 타임라인을 생성하고, AI가 여행 스토리를 작성해주는 iOS 앱입니다.

---

## 주요 기능

### 자동 타임라인 생성
사진을 선택하면 GPS 좌표와 촬영 시간을 분석하여 방문 장소와 이동 경로를 자동으로 파악합니다. 복잡한 입력 없이 사진만으로 여행 기록이 완성됩니다.

### AI 스토리텔링 (BYOK)
OpenAI, Anthropic Claude, Google Gemini 등 원하는 AI 서비스의 API 키를 등록하면, 타임라인을 바탕으로 감성적인 여행 스토리를 생성합니다.

### 지도 시각화
방문한 장소들을 지도 위에 표시하고, 이동 동선을 한눈에 확인할 수 있습니다.

### 공유 & 내보내기
완성된 여행 기록을 이미지나 텍스트로 공유하거나, Markdown/HTML로 내보낼 수 있습니다.

## 특징

| | |
|---|---|
| **100% 온디바이스** | 서버 없이 모든 분석이 기기에서 처리됩니다 |
| **로그인 불필요** | 회원가입, 로그인 없이 바로 사용 |
| **BYOK** | AI 기능은 사용자의 API 키로 동작 (Bring Your Own Key) |
| **완전 무료** | 프리미엄, 구독, 광고 없음 |
| **프라이버시** | 사진은 기기를 떠나지 않습니다 |

---

## 부가 기능

- **지금 뭐해?** - 오늘 찍은 사진으로 빠르게 기록 생성
- **주간 하이라이트** - 이번 주 사진 자동 요약
- **지난 추억** - N년 전 오늘의 기록 되돌아보기
- **보안 잠금** - PIN 또는 Face ID로 앱 보호
- **카테고리 분류** - 여행, 일상, 출장 등으로 기록 정리
