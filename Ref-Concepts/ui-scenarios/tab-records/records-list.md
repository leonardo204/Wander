### 9.10 기록 탭 화면 (SCR-006)

> **v2.2 신규**: 검색 기능, 컨텍스트 메뉴 추가

**기록 탭 레이아웃:**
```
┌─────────────────────────────────┐
│ 9:41                     📶 🔋  │
├─────────────────────────────────┤
│             기록                │
├─────────────────────────────────┤
│ ┌─────────────────────────────┐ │
│ │ 🔍 기록 검색...              │ │
│ └─────────────────────────────┘ │
├─────────────────────────────────┤
│ [전체] [여행] [일상] [주간] ... │
├─────────────────────────────────┤
│                                 │
│ ┌─────────────────────────────┐ │
│ │ [썸네일] 🏝️ 제주도 3박4일    │ │
│ │          2026.01.20~01.23   │ │
│ │          📍 12곳 · 🚗 245km │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ [썸네일] ☕ 성수동 카페 투어   │ │
│ │          2026.01.18         │ │
│ │          📍 4곳 방문         │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 🔒 숨긴 기록 (3)          > │ │
│ └─────────────────────────────┘ │
│                                 │
├─────────────────────────────────┤
│    🏠      📚      ⚙️           │
│    홈      기록    설정          │
└─────────────────────────────────┘
```

**검색 기능:**
- 검색어 입력 시 제목에서 실시간 필터링
- 빈 결과 시: "검색 결과가 없습니다" 표시
- 검색어 지우기 버튼 (X)

**카테고리 필터 칩:**
| 필터 | 설명 |
|------|------|
| 전체 | 모든 기록 표시 |
| 여행 | 여행 카테고리 기록 |
| 일상 | 일상 카테고리 기록 |
| 주간 | 주간 하이라이트 기록 |
| 출장 | 출장 카테고리 기록 |
| 사용자 정의 | 사용자가 추가한 카테고리 |

**기록 컨텍스트 메뉴 (길게 누르기):**
```
┌─────────────────────────────────┐
│  🏝️ 제주도 3박4일               │
├─────────────────────────────────┤
│  ✏️ 제목 수정                   │
│  🏷️ 카테고리 변경               │
│  👁️ 숨기기                      │
│  🗑️ 삭제                        │
└─────────────────────────────────┘
```

**컨텍스트 메뉴 동작:**
| 메뉴 | 동작 |
|------|------|
| 제목 수정 | 텍스트 입력 Alert 표시 |
| 카테고리 변경 | 카테고리 선택 시트 표시 |
| 숨기기 | 인증 후 기록 숨김 처리 |
| 삭제 | 삭제 확인 Alert 표시 (위험 동작) |
