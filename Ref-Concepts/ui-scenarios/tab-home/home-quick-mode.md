## 5. "지금 뭐해?" 퀵 모드

### 5.1 퀵 모드 플로우

```mermaid
flowchart TD
    A[지금 뭐해? 버튼 탭] --> B[퀵모드 사진 선택 시트]

    B --> C{사진 선택 방식}
    C -->|최근 사진에서 선택| D[최근 24시간 사진 표시]
    C -->|카메라로 촬영| E[카메라 실행]
    C -->|갤러리에서 선택| F[전체 갤러리 표시]

    D --> G[사진 선택 완료]
    E --> H[촬영 완료]
    F --> G
    H --> G

    G --> I{선택 사진 수 확인}
    I -->|1-10장| J[빠른 분석 시작]
    I -->|10장 초과| K[10장 제한 안내]
    I -->|0장| L[사진 선택 요청]
    K --> M[처음 10장만 사용 확인]
    M --> J
    L --> B

    J --> N[분석 중 표시]
    N --> O[즉시 결과 카드 생성]

    O --> P{결과 표시}
    P --> Q[결과 카드 + 공유 옵션]
    Q --> R{사용자 선택}

    R -->|공유하기| S[공유 시트]
    R -->|저장하기| T[기록으로 저장]
    R -->|수정하기| U[상세 편집 화면]
    R -->|닫기| V[홈으로 복귀]
```

### 5.4 24시간 내 사진이 없는 경우

```mermaid
flowchart TD
    A[퀵모드 진입] --> B{최근 24시간 사진 확인}

    B -->|사진 있음| C[사진 목록 표시]
    B -->|사진 없음| D[빈 상태 표시]

    D --> E[안내 메시지 표시]
    E --> F{사용자 선택}

    F -->|카메라 열기| G[카메라 실행]
    F -->|갤러리에서 선택| H[전체 갤러리]
    F -->|취소| I[시트 닫기]

    G --> J[촬영 후 계속]
    H --> J
```
