### 6.1 이번 주 하이라이트 플로우

```mermaid
flowchart TD
    A[이번 주 하이라이트 탭] --> B{GPS 포함 사진 스캔}

    B --> C{사진 발견 여부}
    C -->|사진 있음| D[자동 수집 결과 표시]
    C -->|사진 없음| E[빈 상태 안내]

    E --> F{사용자 선택}
    F -->|직접 선택하기| G[사진 선택 화면]
    F -->|취소| H[홈으로 복귀]

    D --> I{사용자 확인}
    I -->|선택 수정| J[사진 추가/제거]
    I -->|이대로 진행| K[분석 시작]

    G --> K
    J --> K

    K --> L[주간 분석 진행]
    L --> M[일자별 요약 생성]
    M --> N[주간 통계 계산]
    N --> O[결과 화면 표시]
```
