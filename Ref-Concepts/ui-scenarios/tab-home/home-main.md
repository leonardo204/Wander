## 3. 메인 화면 (홈)

### 3.1 홈 화면 구조

> **v2.1 변경사항**: "여행 기록 만들기"를 FAB로 분리, 퀵 액션 2개로 축소

```mermaid
graph TB
    subgraph "홈 화면 구조"
        A[Status Bar]
        B[헤더 영역]
        C[퀵 액션 카드 2열]
        D[최근 기록 섹션]
        E[탭 바]
        F[FAB: 여행 기록 만들기]
    end

    A --> B
    B --> C
    C --> D
    D --> E
    F -.->|우하단 플로팅| E

    subgraph "퀵 액션 카드 상세"
        C --> G[지금 뭐해?]
        C --> H[돌아보기]
    end
```

### 3.2.1 돌아보기

> **v2.2 변경사항**: 홈 화면에서 바로 시트로 열림 (드롭다운 제거)

**동작:**
- 돌아보기 카드 탭 → LookbackView 시트 표시
- 시트 내에서 기간 선택 후 분석 진행

**기간 옵션 (LookbackView 내 선택):**
| 옵션 | 설명 |
|------|------|
| 이번 주 | 최근 7일간 촬영한 사진 |
| 지난 주 | 7~14일 전 촬영한 사진 |
| 이번 달 | 최근 30일간 촬영한 사진 |
| 올해 | 올해 촬영한 사진 |

**빈 상태 처리:**
- 선택한 기간에 사진이 없으면: "선택한 기간에 촬영한 사진이 없어요"
- 다른 기간 제안: "다른 기간을 선택해 보세요" 버튼 표시

### 3.3 홈 화면 상호작용

```mermaid
flowchart TD
    subgraph "홈 화면 액션"
        A[홈 화면]

        A -->|FAB 탭| B[사진 선택 화면]
        A -->|지금 뭐해? 탭| C[퀵모드 사진 선택]
        A -->|돌아보기 탭| D[기간별 사진 자동 수집]
        A -->|돌아보기 드롭다운| D1[기간 선택 시트]
        A -->|최근 기록 탭| E[기록 상세 화면]
    end

    subgraph "FAB 동작"
        B --> B1{사진 권한 확인}
        B1 -->|권한 있음| B2[사진 그리드 표시]
        B1 -->|권한 없음| B3[권한 요청 안내]
    end

    subgraph "빈 상태 처리"
        A -->|기록 없음| G[빈 상태 화면]
        G --> H[FAB로 첫 기록 유도]
    end
```
