## 7. 결과 화면 및 상세 보기

### 7.1 결과 화면 구조 (SCR-010)

> **v2.2 변경사항**: Wander Intelligence 섹션 추가 (여행 점수, 여행자 DNA, 인사이트, 스토리 미리보기)

```mermaid
graph TB
    subgraph "결과 화면 구조"
        A[지도 섹션]
        B[통계 섹션]
        C[Wander Intelligence 섹션]
        D[타임라인 섹션]
        E[AI 스토리 섹션]
        F[액션 버튼 영역]
    end

    A --> B
    B --> C
    C --> D
    D --> E
    E --> F

    subgraph "지도 섹션 기능"
        A --> A1[핀 표시]
        A --> A2[경로 라인]
        A --> A3[탭하여 확대]
    end

    subgraph "Wander Intelligence"
        C --> C1[여행 점수 카드]
        C --> C2[여행자 DNA 카드]
        C --> C3[인사이트 미리보기]
        C --> C4[스토리 미리보기]
    end

    subgraph "타임라인 섹션 기능"
        D --> D1[일자별 그룹]
        D --> D2[장소별 카드]
        D --> D3[사진 썸네일]
        D --> D4[활동 라벨]
    end

    subgraph "액션 버튼"
        F --> F1[공유하기]
        F --> F2[저장하기]
    end
```

### 7.3 타임라인 상호작용

```mermaid
flowchart TD
    subgraph "타임라인 상호작용"
        A[타임라인 섹션]

        A -->|장소 카드 탭| B[장소 상세 시트]
        B --> B1[주소 전체 표시]
        B --> B2[사진 전체 보기]
        B --> B3[지도에서 보기]
        B --> B4[메모 추가]

        A -->|사진 썸네일 탭| C[사진 뷰어]
        C --> C1[좌우 스와이프]
        C --> C2[확대/축소]
        C --> C3[메타데이터 보기]

        A -->|길게 누르기| D[편집 모드]
        D --> D1[순서 변경]
        D --> D2[사진 제거]
        D --> D3[장소 병합]
    end
```

### 7.5 타임라인 편집 화면 (SCR-012)

```mermaid
flowchart TD
    A[편집 버튼 탭] --> B[편집 모드 진입]

    B --> C{편집 액션}

    C -->|장소 카드 드래그| D[순서 변경]
    D --> E[순서 저장]

    C -->|사진 선택| F[선택 모드]
    F --> G{액션 선택}
    G -->|삭제| H[사진 제거]
    G -->|이동| I[다른 장소로 이동]

    C -->|장소 병합| J[두 장소 선택]
    J --> K[하나로 합치기]

    C -->|장소명 수정| L[텍스트 입력]
    L --> M[이름 저장]

    C -->|활동 라벨 변경| N[라벨 선택기]
    N --> O[라벨 적용]

    C -->|메모 추가| P[메모 입력]
    P --> Q[메모 저장]

    B -->|완료 버튼| R{변경사항 확인}
    R -->|저장| S[편집 완료]
    R -->|취소| T[변경 취소 확인]
```
